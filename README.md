# 📺 电视直播源项目 - 优化分类版

此项目会自动收集电视直播源，并按9大分类整理。

## 📊 统计信息
- **更新时间**: 2026-01-31 11:15:34
- **频道总数**: 2577
- **数据源**: 2 个

1. **央视** - 中央电视台及CCTV系列频道
2. **卫视** - 各省市卫星电视台
3. **地方台** - 地方新闻、都市、民生频道
4. **少儿台** - 少儿、卡通、动漫频道
5. **综艺台** - 综艺、娱乐、文艺频道
6. **港澳台** - 香港、澳门、台湾地区频道
7. **体育台** - 体育赛事、足球、篮球等频道
8. **影视台** - 电影、影院、影视剧频道
9. **其他台** - 未分类的频道

## 🚀 快速部署

1. **克隆或Fork此仓库**
2. **等待GitHub Actions自动运行**
3. **下载生成的播放列表文件**

## 📁 项目结构
├── .github/workflows/update-live-sources.yml # GitHub Actions工作流
├── scripts/collect_sources.py # 采集脚本(优化分类版)
├── sources.txt # 源列表
├── live_sources.m3u # 自动生成：完整直播源
├── channels.json # 自动生成：频道数据
├── index.html # 自动生成：网页界面
└── categories/ # 自动生成：9大分类直播源


## ⚙️ 配置

编辑 `sources.txt` 文件可以添加更多直播源URL。

## 📅 更新频率

每天自动更新一次，保持直播源最新。
